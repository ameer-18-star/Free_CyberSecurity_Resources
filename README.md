# 🛡️ Free Cybersecurity & Information Security Resources

Welcome to Free Cybersecurity Resources—a curated, community-driven collection of no-cost learning materials designed to help you launch or elevate your career in cybersecurity. Whether you're mastering networking basics, training for certifications, hunting bugs, or analyzing threats on the front lines, this repository brings together top-tier courses, hands-on practice platforms, cheatsheets, and tools into one structured, zero-cost roadmap.

## 📌 Repository Overview

Breaking into cybersecurity shouldn't come with a high price tag. **Free Cybersecurity Resources** serves as a central hub designed for students, self-taught security enthusiasts, and seasoned professionals looking to sharpen their skills without spending thousands on training.

Whether you're aiming for a role in **Defensive Security (Blue Team)**, **Offensive Security (Red Team)**, **Cloud Security**, or **Bug Bounty Hunting**, this repository provides a zero-cost path forward.

## 🔰 Fundamentals & Core Skills

Before diving into advanced hacking techniques or threat hunting, master these foundational building blocks:

### 🌐 Computer Networking
* **[Professor Messer's Network+ Course](https://www.professormesser.com/networkplus/n10-008/n10-008-training-course/)** – Industry-standard free video course covering core networking concepts.
* **[FreeCodeCamp Computer Networking Course](https://www.youtube.com/watch?v=IPvYjXCsTg8)** – A full 4-hour video guide explaining IP addressing, subnets, and protocols.
* **[Packet Tracer by Cisco Networking Academy](https://www.netacad.com/courses/packet-tracer)** – Free software and introductory course to build virtual network topologies.

### 🐧 Linux & Systems Administration
* **[OverTheWire: Bandit](https://overthewire.org/wargames/bandit/)** – Interactive wargame designed for beginners to learn basic Linux commands and privilege escalation.
* **[Linux Journey](https://linuxjourney.com/)** – Free structured site covering Linux command line, text manipulation, process management, and networking.

### 🐍 Programming & Automation
* **[Automate the Boring Stuff with Python](https://automatetheboringstuff.com/)** – Learn Python from scratch with a focus on automation scripts.
* **[Bash Scripting Tutorial for Beginners](https://ryanstutorials.net/bashscriptingtutorial/)** – Essential guide for shell scripting to automate tasks in Linux.

## 🛡️ Defensive Security & Incident Response (Blue Team)

Defend systems, analyze threat actors, and investigate security breaches.

* **[LetsDefend](https://letsdefend.io/)** – Hands-on SOC analyst simulation platform designed to investigate real-world alerts and malware.
* **[CyberDefenders](https://cyberdefenders.org/)** – Blue team-focused practice platform with labs on DFIR (Digital Forensics and Incident Response), Memory Forensics, and Threat Hunting.
* **[The DFIR Report](https://thedfirreport.com/)** – Real-world case studies detailing actual intrusion scenarios, threat actor TTPs, and detection mechanisms.
* **[Splunk Education Free Courses](https://www.splunk.com/en_us/training/free-courses/overview.html)** – Learn log analysis and SIEM fundamentals with Splunk's free training modules.

## ⚔️ Offensive Security & Penetration Testing (Red Team)

Simulate cyberattacks to identify vulnerabilities before malicious actors exploit them.

* **[PortSwigger Web Security Academy](https://portswigger.net/web-security)** – The gold standard for free web application penetration testing training with interactive labs.
* **[HackTricks](https://book.hacktricks.xyz/)** – A comprehensive wiki for penetration testing methodologies, privilege escalation, and active directory exploitation.
* **[PayloadsAllTheThings](https://github.com/swisskyrepo/PayloadsAllTheThings)** – A vast repository of copy-paste payloads and bypass techniques for web security.
* **[ATT&CK Matrix by MITRE](https://attack.mitre.org/)** – Globally accessible knowledge base of adversary tactics, techniques, and procedures (TTPs).

## 🐛 Bug Bounty & Web Application Security

Discover security flaws in web applications and earn rewards responsibly.

* **[Hacker101](https://www.hacker101.com/)** – Free video lessons and CTF challenges curated by HackerOne.
* **[Bugcrowd University](https://www.bugcrowd.com/hackers/bugcrowd-university/)** – Free educational modules covering reconnaissance, web vulnerabilities, and reporting.
* **[OWASP Top 10](https://owasp.org/www-project-top-ten/)** – The critical baseline document outlining the top 10 web application security risks.

## ☁️ Cloud Security & DevSecOps

Learn to secure AWS, Azure, GCP, and modern containerized environments.

* **[flAWS & flAWS2](http://flaws.cloud/)** – Interactive hands-on challenges to learn AWS security flaws and misconfigurations.
* **[CloudGoat by Rhino Security Labs](https://github.com/RhinoSecurityLabs/cloudgoat)** – Vulnerable-by-design AWS environment generator for practicing cloud pentesting.
* **[Kubernetes Security Guidance by NSA/CISA](https://www.cisa.gov/resources-and-tools/resources/kubernetes-hardening-guidance)** – Official recommendations for securing containerized environments.
